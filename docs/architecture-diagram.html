Spring AI Alibaba — Technical Architecture

Whole-repo · Technical profile · Java 17 · Spring Boot 3.5.x (admin 3.3.6) · Spring AI 1.1.x

Examples (examples/*)

  • • chatbot — quickstart ReactAgent chat
  • • multimodal — vision + TTS via qwen-vl-plus
  • • voice-agent — realtime STT → Agent → TTS over WebSocket
  • • deepresearch — iterative research with MCP / Jina
  • • multiagent-patterns — pipeline · routing · supervisor · workflow · subagent · skills · handoffs ×2
  • • documentation — A2A demo, remote MCP recipes
  • • agentscope/handoffs — AgentScope DSL integration

Framework & Servers

  • • spring-ai-alibaba-graph-core — stateful graph runtime, persistence (PG / MySQL / Oracle / Mongo / Redis / file)
  • • spring-ai-alibaba-agent-framework — built-in patterns (Sequential/Parallel/Routing/Loop/React), A2A, MCP, GraalVM polyglot
  • • spring-ai-alibaba-sandbox — code-execution sandbox (AgentScope Runtime 1.0.2)
  • • spring-ai-alibaba-studio — embedded chat / debug UI · /graph-studio/api/** · /run_sse · /resume_sse
  • • spring-ai-alibaba-admin (Spring Boot 3.3.6) — server-start · server-core · server-runtime · server-openapi

Spring Boot Starters & BOM

  • • starter-a2a-nacos — Agent-to-Agent over Nacos discovery (A2A SDK 0.2.5)
  • • starter-config-nacos — hot-reload of prompts / tools / models (Nacos 3.1)
  • • starter-graph-observation — OpenTelemetry 1.38 spans for graph nodes
  • • starter-builtin-nodes — reusable ChatModel · Tool · Routing · Loop · Subgraph nodes
  • • spring-ai-alibaba-bom — Spring AI 1.1.2 · DashScope 2.15.1 · MCP 0.14 · Jackson 2.18 · OTel 1.38

Data Stores

  • • MySQL 9.x — admin OLTP, 27 tables across agentscope-schema + admin-schema
  • • Redis (Redisson 3.42) — sessions, cache, optional graph checkpoint backend
  • • Elasticsearch — vector store (Knowledge Base / RAG / DocumentChunk)
  • • PostgreSQL · Oracle · MongoDB · File · In-memory — pluggable graph-core checkpoint savers
  • • H2 — test-scope only

Middleware

  • • Nacos 3.1 — config center + service discovery (A2A registry)
  • • Aliyun OSS — document & knowledge-base file storage (S3-compatible)
  • • RocketMQ 5.0.7 — async document indexing pipeline (topic: topic_saa_studio_document_index)
  • • OTLP collector — receives tracing/metric export (Jaeger / Aliyun ARMS / Datadog)
  • • Druid 1.2.20 — JDBC connection pool with stat / wall / slf4j filters

External APIs

  • • Alibaba DashScope — primary model provider (chat, embeddings, vision, TTS, realtime audio)
  • • OpenAI · Anthropic · DeepSeek · ZhipuAI — pluggable via spring-ai-starter-model-*
  • • Ollama — local inference over HTTP
  • • MCP servers (Jina Reader mcp.jina.ai, custom tool servers) — JSON-RPC over stdio / streamable HTTP
  • • Remote A2A peers — distributed agents discovered via Nacos
